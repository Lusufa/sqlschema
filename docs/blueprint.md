# **App Name**: SQL Genius

## Core Features:

- SQL Query Generation: Generates SQL queries from natural language questions based on the provided database schema using AI.
- Schema Input: Allows users to input database schema definitions.
- Natural Language Input: Provides a text input field for users to enter their natural language queries.
- SQL Output Display: Displays the generated SQL query in a formatted text box.
- Query Execution Tool: Tests generated query tool to a database specified by the user.

## Style Guidelines:

- Primary color: Black (#000000) to evoke trust and technical expertise.
- Background color: White (#FFFFFF) to provide a clean and unobtrusive backdrop.
- Accent color: Gray (#808080) to highlight key interactive elements and provide visual interest.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text. This combination provides a balance of technical precision and readability.
- Code font: 'Source Code Pro' (monospace) for displaying SQL queries.
- Use minimalist line icons to represent database elements, input fields, and actions.
- Divide the interface into distinct sections for schema input, natural language query input, and SQL output, using a clear grid system.
